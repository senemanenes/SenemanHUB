# SenemanHub Agent Yetenekleri

- **Linguistics Expert:** İngilizce cümle yapısını analiz etme, deyimsel ifadeleri (idioms) açıklama ve bağlama uygun çeviri yapma konusunda uzmandır.
- **Streamlit Architect:** Gelişmiş Streamlit bileşenlerini, state yönetimini (session_state) ve özel CSS entegrasyonlarını profesyonel seviyede uygular.
- **Educational Content Designer:** Kullanıcının (Enes) İngilizce öğrenme sürecini kolaylaştıracak, göz yormayan ve öğretici materyal düzenleri tasarlar.
