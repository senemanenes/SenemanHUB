# 🏛️ SenemanHUB | Language Intelligence Vault


---

## 📜 Proje Hakkında (The Vision)
SenemanHUB, sadece bir kelime ezberleme aracı değil, aynı zamanda **Gemini AI** destekli bir dil asistanıdır. Kullanıcıların kendi kelime hazinelerini yönetmelerine, okuma yeteneklerini geliştirmeye ve cümle yapılarını analiz etmelerine olanak tanır.

## ✨ Temel Özellikler
* **🤖 AI Destekli Öğrenme:** Google Gemini API entegrasyonu ile kelimelerin bağlamsal anlamlarını ve örnek kullanımlarını anında analiz eder.
* **📊 Google Sheets Entegrasyonu:** Verileri güvenli ve erişilebilir bir şekilde Google Sheets üzerinde depolar (Canlı veri senkronizasyonu).
* **📱 PWA Desteği:** Mobil cihazlara bir "mühür" ikonuyla yüklenebilir, tarayıcı bağımsız bir uygulama deneyimi sunar.


## 🛠️ Teknik Şantiye (Tech Stack)
* **Dil:** Python 3.x
* **Arayüz:** [Streamlit](https://streamlit.io/)
* **Zeka:** Google Gemini Pro API
* **Veritabanı:** Google Sheets API (gspread)
* **Barındırma:** Streamlit Cloud
